Example of Preformatted Text

The text inside the <pre> tag is displayed in a fixed-width font and preserves both spaces and line breaks.

Example Program

Below is an example of a simple Python program displayed within a <pre> tag:


# Simple Python Program
def greet(name):
    print(f"Hello, {name}!")

# Call the function
greet("World")

Explanation

The above code defines a function greet that takes a parameter name and prints a greeting message. The function is then called with the argument "World", which results in the output:


Hello, World!
